## 🌟 主な掲載プロジェクト

### 1. 🛡️ 詐欺被害防止シミュレーションシステム
AIを活用した、詐欺手口を擬似体験できる教育システム。
- **担当範囲:** AIの対話ロジック、選択肢分岐のコーディング
- **こだわり:** 5択形式のUIを採用し、ゲーム感覚で学べるように設計。

### 2. 📢 誘拐防止！大声採点システム
小学生向けの防犯訓練ツール。
- **担当範囲:** 企画、システムロジックの検討
- **こだわり:** 採点・ランキング機能を実装し、楽しみながら防犯意識を高める工夫。
